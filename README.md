# 💳 AI-Powered Credit Card Fraud Detection

## 📌 Overview

This project uses Machine Learning to detect fraudulent credit card transactions.

It is built using Python, Streamlit, and a Random Forest Classifier with an interactive dashboard for visualization and prediction.

---

## 🚀 Features

- 💳 Credit Card Fraud Detection
- 📊 Interactive Dashboard
- 📈 Exploratory Data Analysis (EDA)
- 🔥 Confusion Matrix
- 📉 ROC Curve
- 🌟 Feature Importance Analysis
- 🤖 Prediction Lab
- 📋 Real-time Transaction Analysis
- 📥 Download Prediction Results

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- Scikit-learn
- Joblib
